chicken



Le formulaire a l'identificateur crLecture. Ce nom sera utilisé
pour accéder au formulaire dans le code JavaScript.
Les trois premiers contrôles du formulaire sont des contrôles cachés
qui sont en fait des informations nécessaires à WebMail
Les contrôles du formulaire qu'on veu
Qu'est-ce qui arrive à une page contenant du JavaScript dans le cas où le Javascript est désactivé? Habituellement rien,
simplement que le code JavaScript n'est pas exécuté.
Cependant il est possible de détecter le fait que le lecteur
a désactivé le JavaScript. On pourrait alors souhaiter informer le lecteur que la page
contient des fonctionnalités additionnelles accessibles en activant le JavaScript.
Ça se fait au moyen de l'élément noscript.
